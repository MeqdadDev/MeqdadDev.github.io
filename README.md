# Meqdad A. Darwish
### Back-End Developer 💻 | Founder @ <a href="https://bunyanflow.com" target="_blank">Bunyan Flow</a> 🏗️ | Blogger 🖋️ | Roboticist 🤖 | Reader 📚

### Summary:
I'm a back-end developer — building my own product, and open to my next role.

Bunyan Flow is a bilingual (Arabic/English) property management platform I founded. I'm building the MVP end to end — Java, Spring Boot, PostgreSQL — owning everything from requirements analysis and database design to RESTful services, KPI dashboards, and CI/CD with GitHub Actions. Solo founder means full SDLC, full accountability. No handoffs. If it's broken, it's mine to fix.

Before that: production back-end work at LogesTechs, where I built and maintained a logistics system processing 30k+ daily data entries — customer wallet features, audit logging, real-time reporting. Java, Spring Boot, MySQL, Agile teams.

I also spent years on the education side of tech — training 2,000+ learners, from students to Ministry of Education engineers, and leading ToT programmes in robotics and AI. That chapter taught me something most developers learn late: if you can't explain a system simply, you don't understand it yet. It made me a better engineer, and it's why building products people actually understand comes naturally to me.

Open source is where I sharpen the saw: my Python packages on PyPI have passed 50k+ downloads. I write technical articles on databases (SQL and NoSQL), clean code, AI, and programming — with Java and Spring Boot topics coming next.

Off the keyboard: volunteering, books. Half my reading is technical and the other half is everything else, because the best engineers I know are readers first.

### 💻 Detectors World (Python Package)

#### Tags: _Computer Vision, MediaPipe, Detection_
#### Links: <a href="https://github.com/MeqdadDev/detectors-world" target="_blank">GitHub</a> | <a href="https://pypi.org/project/detectors-world" target="_blank">PyPI</a>

Detectors World is a Python package created for building Computer Vision applications in an easy and handy way, built on top of the MediaPipe framework.

The available detectors in Detectors World package are:
- Hand Detection
- Face Detection
- Pose Detection
- Face Mesh Detection


### 💻 Teachable Machine (Python Package)

#### Tags: _Computer Vision, OpenCV, Classification_
#### Links: <a href="https://meqdaddev.github.io/teachable-machine/" target="_blank">Documentation</a> | <a href="https://github.com/MeqdadDev/teachable-machine" target="_blank">GitHub</a> | <a href="https://pypi.org/project/teachable-machine/" target="_blank">PyPI</a>

A Python package designed to simplify the integration of exported models from Google's <a href="https://teachablemachine.withgoogle.com/" target="_blank">Teachable Machine</a> platform into various environments. This tool was specifically crafted to work seamlessly with Teachable Machine, making it easier to implement and use your trained models.


### 💻 PixCraft (Python Web App)
#### Tags: _Image Processing, Flask, Computer Vision, OpenCV_
#### Link: <a href="https://github.com/MeqdadDev/PixCraft" target="_blank">GitHub</a>

PixCraft (Unleash the Power of Pixels) is a powerful image processing and computer vision application that allows users to explore and manipulate digital images effortlessly. With its intuitive interface, users can upload images or access their device's camera, and then apply a wide range of image processing operations like grayscale conversion, HSV color space transformation, edge detection, and more.

### 💻 Teachable Machine Lite (Python Package)

#### Tags: _Computer Vision, OpenCV, Classification, Raspberry Pi, Robotics_
#### Links: <a href="https://meqdaddev.github.io/teachable-machine-lite/" target="_blank">Documentation</a> | <a href="https://github.com/MeqdadDev/teachable-machine-lite" target="_blank">GitHub</a> | <a href="https://pypi.org/project/teachable-machine-lite/" target="_blank">PyPI</a>

A lightweight Python package optimized for integrating exported models from Google's [Teachable Machine Platform](https://teachablemachine.withgoogle.com/) into robotics and embedded systems environments. This streamlined version of [Teachable Machine Package](https://github.com/MeqdadDev/teachable-machine) is specifically designed for resource-constrained devices, making it easier to deploy and use your trained models in embedded applications. With a focus on efficiency and minimal dependencies, this tool maintains the core functionality while being more suitable for robotics and IoT projects.
